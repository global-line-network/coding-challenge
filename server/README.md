## Requirements

- [MongoDB](https://www.mongodb.com/download-center)
- [Redis](https://redis.io/download)
- [Node v7.6+](https://nodejs.org/en/download/package-manager/)

Node dependencies:

```
npm install
```

## Getting started

```
npm run start-dev
```
